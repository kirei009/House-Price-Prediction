# House Price Prediction

## Project Overview

This project aims to predict house prices using machine learning techniques. The dataset contains various property features such as area, number of bedrooms, bathrooms, parking availability, furnishing status, and other amenities. The goal is to analyze the data, build predictive models, evaluate their performance, and identify the factors that most strongly influence house prices.

## Dataset

Dataset Used: Housing Prices Dataset (Housing.csv)

## Tasks Performed

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding
4. Train-Test Split (80/20)
5. Linear Regression Model Training
6. Random Forest Regressor Model Training
7. Model Evaluation using MAE, RMSE, and R² Score
8. Data Visualization
9. Insights and Summary

## Tools and Libraries Used

* Python 3

* NumPy

## Visualizations

* House Price Distribution Histogram
* Correlation Heatmap
* Actual vs Predicted Price Scatter Plot

## Conclusion

The project successfully demonstrates the use of machine learning models for predicting house prices. The analysis highlights the importance of property size, amenities, and location-related factors in determining house value.
